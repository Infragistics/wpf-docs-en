= Styling Points for xamMap

== Topic Overview

=== Purpose

This topic provides designers styling points for the link:{ApiPlatform}controls.maps.xammap.v{ProductVersion}~infragistics.controls.maps.xammap.html[xamMap]™ control.

== Styling Points
