=== Introduction

This section contains topics covering the  _xamBusyIndicator_™ control for WPF.

The  _link:{RootAssembly}{ApiVersion}~infragistics.controls.interactions.xambusyindicator.html[xamBusyIndicator]_  is a ContentControl that provides visual notification that a long-running activity has started as well as reports the actual progress.
