= xamSparkline

=== Introduction

The link:{ApiPlatform}controls.charts.xamsparkline{ApiVersion}.html[xamSparkline]™ is a lightweight charting control that can render the following chart types:

* Line
* Area
* Column
* Win/Loss

It is intended for rendering in a small scale layout such as a grid Cell. It can also be rendered stand alone. The following previews illustrate the four Sparkline types.
